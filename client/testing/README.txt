Script vari per test relativi a GameSpeed, GoalScore e Statistics

1)ParseGameSpeed -> test relativi alla velocità di gioco incrementata (1x, 2x ... 12x)

2)ParseGoalScore -> test relativi al goal score (20, 40, 80, 160)

3)ParseStatistics -> test per vedere le statistiche relative a time kill, distanza media, kill streak e precisione.
