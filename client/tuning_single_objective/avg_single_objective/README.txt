Contiene il calcolo della media di tutte le run effettuate(10) con l'algoritmo genetico a singolo obiettivo.

population_cluster -> contiene tutte le popolazioni finali delle 10 run
best -> contiene tutti gli individui delle popolazioni finali con fitness pari a max (3)

avg_of_avg -> contiene la media e la standard deviation per generazione di tutte le run

cluster -> contiene il cluster effettuato su tutte le popolazioni finali
